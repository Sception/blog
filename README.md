# blog
vuepress搭建的个人博客



# 开发环境
```javascript
git clone https://github.com/Sception/blog.git

git checkout dev

yarn install

yarn run dev

```

# 生产环境
```javascript
git clone https://github.com/Sception/blog.git

git checkout go-pages

```


# 部署
```javascript
yarn run deploy

```
